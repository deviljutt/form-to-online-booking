CF7 to Calendly integration with parameters and values

== Description ==
You can integrate you Calendly meeting URL with contact form 7, After submission of your contact form 7 you will redirect to Calendly meeting link auto-populated fields with your contact form 7 form details :)


== Installation ==
Upload the plugin files to the ‘/wp-content/plugins/’ directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the \\’Plugins\\’ screen in WordPress
Go the Contact form 7 and open Calendly Integration tab to configure the plugin

== Frequently Asked Questions ==
* Can we use this plugin without cf7
No, This plugin required cf7 installation

* Can we deactivate it to some forms
Yes , the Default option is deactivated if you want to activate if you have to got the the \"Calendly Integration\" tab in your form
